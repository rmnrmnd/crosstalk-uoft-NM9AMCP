# Models Directory

This folder is used to deposit your best trained and serialized models.

## Conventions

- Save your best-performing model here after training using a serialization library like `pickle` or `joblib`.
- Default naming recommendation: `best_model.pkl` (or `best_model.joblib`).
- When submitting your repository, make sure your final serialized model file is committed and pushed so that it can be retrieved for evaluation/grading.
